Mga Madalas Itanong (FAQ)

Bakit kailangan i-enable ang Desktop site?

Ang mobile version ng Facebook ay may limitadong functionality. Ang desktop version ang may kumpletong features na kailangan ng extension para gumana nang maayos.

Hindi ko makita ang C3C FBState Utility sa menu

Subukan ang mga sumusunod:

  1. Siguraduhing naka-install nang tama ang extension
  2. I-refresh ang Facebook page o i-restart ang browser
  3. Tiyaking naka-Desktop site mode
  4. Check kung naka-enable ang extension sa Kiwi Browser settings
Safe ba ang paggamit ng FBState Utility?

Oo, safe naman basta:

  • Huwag ibahagi ang iyong fbstate sa iba
  • Gamitin lamang ang extension sa sariling device
  • I-logout agad kung gagamit sa public computer
Pwede bang gamitin sa ibang browser?

Ang C3C FBState Utility ay designed para sa Chrome-based browsers. Sa Android, ang Kiwi Browser lang ang kilalang browser na pwedeng mag-install ng Chrome extensions.
