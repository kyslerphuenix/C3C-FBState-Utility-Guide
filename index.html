Frequently Asked Questions (FAQ)

Why do I need to enable Desktop site?

The mobile version of Facebook has limited functionality. The desktop version has all the features needed for the extension to work properly.

I can't see C3C FBState Utility in the menu

Try the following:

  1. Make sure the extension is properly installed
  2. Refresh the Facebook page or restart the browser
  3. Make sure you're in Desktop site mode
  4. Check if the extension is enabled in Kiwi Browser settings
Is using FBState Utility safe?

Yes, it's safe as long as:

  • Don't share your fbstate with others
  • Only use the extension on your own device
  • Logout immediately if using a public computer
Can I use it with other browsers?

C3C FBState Utility is designed for Chrome-based browsers. On Android, Kiwi Browser is the only known browser that can install Chrome extensions.
